<h2 align="left">🎬 My Featured Projects</h2>

<details>
<summary>Click to view technical details: <b>MovieDB - PostgreSQL Relational Database System</b></summary>
<br>

<p align="left">A comprehensive relational database (RDBMS) designed from scratch for a social movie platform (Letterboxd alternative). The core focus is on data integrity, user interactions, and database-level automation using advanced SQL and PL/pgSQL.</p>

### 🚀 Key Technical Features
    
* **PL/pgSQL Recommendation Logic:** Analyzes user history and favorite genres to suggest new movies (e.g., `film_oner_favori_ture_gore` function).
* **Data Integrity & Automation (Triggers):**
  * `spam_engelleme`: Prevents multiple reviews or ratings for the same movie.
  * `clean_user_data`: Automatically formats usernames and emails during registration.
* **Analytical Reporting (Views):**
  * `yonetmen_karnesi`: Dynamically lists director performance and average ratings.
  * `pasif_kullanici_alarmi`: Identifies users with no action in the last 90 days.
